# Praktická časť práce SOČ
Táto stránka vznikla ako praktická časť mojej práce SOČ, v ktorej som sa zameriavala na tému "Tvorba jednoduchých webových stránok a ich využitie". Stránka je určená študentom stredných škôl a poskytuje im prehľad a krátke zhrnutie diel povinného čítania.

- Jedná sa o môj vôbec prvý samostatný projekt v oblasti vývoja webov. Stránka bola vytvorená v čistom HTML a CSS.
